# slidet-oamk

Slides for OAMK presentation

## Installation

Download Leiningen from https://leiningen.org/

## Usage

Run by executing 

$ lein repl 

Run `(-main)` to start the backend server

Run 

`(require '[figwheel.main.api :as fig])`

`(fig/start "dev")`

to start the frontend server

Navigate to http://localhost:3005
